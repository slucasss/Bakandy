# Bakandy (C++ - SDL)

A shoot'em up made in C++ with SDL lib in 2 weeks

![Language](https://img.shields.io/badge/language-C%2B%2B-blue)
![Platform](https://img.shields.io/badge/platform-Windows-lightgrey)
![Status](https://img.shields.io/badge/status-Completed-green)

The objective is to survive through enemies waves and defeat the boss. 

## Features
- Controlable spaceship
- Special skill for the player
- Different enemies with different patterns
- Boss with different skills
- Customizable level (in a level.txt)

## Controles

- Z -> Move up
- S -> Move down
- Q -> Move left
- D -> Move right
- Space -> Shoot classic bullet
- E -> Shoot homing bullet

## Installation

1. Clone the repository:
   git clone https://github.com/slucasss/Bakandy.git

3. Get the .sln for visual studio
```bash
  .\SolutionGenerator.exe -make ./
```

3. Open it in visual studio

4. Compile

5. Paste the dlls in Debug file

6. Run
